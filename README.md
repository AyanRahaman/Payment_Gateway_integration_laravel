# Razorpay Payment Gateway Integration in Laravel

This project demonstrates how to integrate a payment gateway into a Laravel application. It provides a simple implementation of an online payment system where users can select products and complete payments securely.

The project is built using the Laravel framework and integrates a payment gateway to process transactions. It includes features such as user authentication, product listing, and secure payment handling.

## Features

* Laravel-based payment gateway integration
* User registration and login system
* Product listing with pricing
* Secure payment processing
* Payment success confirmation page
* Simple and clean UI using Bootstrap

## Technologies Used

* Laravel
* PHP
* MySQL
* Bootstrap
* JavaScript
* Razorpay Payment Gateway (for payment processing)

## Purpose of the Project

The main goal of this project is to demonstrate how payment gateway integration works in Laravel. It is useful for beginners who want to learn how to implement online payment systems in web applications.

## How to Run the Project

1. Clone the repository
2. Install dependencies using `composer install`
3. Configure the `.env` file
4. Run database migrations and seeders
5. Start the development server using `php artisan serve`

This project can be used as a learning reference for implementing payment gateway integration in Laravel applications.
